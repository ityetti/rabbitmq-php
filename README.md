# Simple PHP Script for RabbitMQ

This is a simple AMQP send-receive client for RabbitMQ.

### Installation
```
git clone git://github.com/ityetti/rabbitmq-php.git
cd rabbitmq-php
composer install
```

### Run consumer (receiver)
`php receive.php`

### Run publisher (sender):
`php send.php`
